# NIMA

**Nearby Instant Meet App**

**Tagline:** Meet safely, connect silently.  
**Slogan:** Connecting people with consent.

This is the Flutter foundation project for NIMA.

## Version
NIMA Flutter Project v1.0

## Current Screens
- Splash
- Onboarding
- Phone Login UI
- OTP UI
- Create Profile
- Home
- Nearby Radar
- Hi Requests
- Chat
- Notifications
- Premium
- Settings

## Mobile-only build workflow
1. Upload these files to GitHub repository `nima-app`.
2. Connect repository to Codemagic.
3. Run the `android-debug` workflow.
4. Download the generated APK.
